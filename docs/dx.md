# Developer experience

## Preinstalled applications

 - Zed - greate simple but powerful editor with AI, git support, themes, extentions and more
 - VSCodium - like VSCode but without propertiary parts and telemetry
 - Restic and RClone
 - Waydroid - for running android apps
 - scrcpy - for mirroring android screen via ADB
 - docker and docker-compose
 
## Virtualization

DX image comes with Gnome Boxes for virtualization. It's greate and simple app for virtual machines. If you want something more andvanced, quickemu, quickget and qemu are also preinstalled.

## Programing languages

Spacefin comes with python and java preinstalled. If you want other language, you can install it via Spacefin-cli (recommended) or Brew.
